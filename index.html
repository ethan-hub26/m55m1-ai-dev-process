NuEdgeWise

BSP 기본 샘플보다 더 넓은 머신러닝 툴과 예제를 다루는 확장형 저장소입니다. README 기준으로는 얼굴/포즈 인식, 랜드마크, YOLO 객체 검출 같은 추가 샘플을 참고하는 경로에 해당합니다.

  • BSP 저장소가 직접 담지 않는 추가 ML 샘플과 툴 참조 경로입니다.
  • 현재 올라간 한국어 가이드들은 NuEdgeWise 사용법 자체를 설명하는 문서는 아닙니다.
  • 따라서 본 가이드 모음은 NuEdgeWise 전용 문서라기보다 M55M1 예제 가이드 모음으로 보는 편이 정확합니다.
